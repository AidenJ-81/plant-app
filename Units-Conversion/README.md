# 단위 변환 Tool · Units Conversion

> 공학 실무를 위한 단일 파일 단위 환산 도구  
> SI(ISO 80000) · ASME · API 표준 환산 계수 기반 · 15개 물리량 카테고리

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| **15개 카테고리** | 길이·면적·부피·질량·밀도·힘·압력·온도·에너지·동력·속도·체적유량·질량유량·동점도·동점도(운동) |
| **실시간 변환** | 값 입력 즉시 모든 단위로 동시 환산 |
| **피벗 전환** | 결과 카드 클릭 → 해당 단위를 새 기준 단위로 전환 + 클립보드 복사 |
| **표시 옵션** | 유효숫자 3~10 선택, 자동/지수 표기법 전환 |
| **도움말 모달** | 앱 사용법·환산 방식·주의사항 내장 |
| **도움말 편집 & 다운로드** | 도움말을 화면에서 직접 수정 후 index.html로 저장 |
| **서버 불필요** | 단일 HTML 파일, 브라우저에서 바로 실행 |

---

## 사용 방법

1. 좌측 **변환 카테고리** 그리드에서 물리량을 선택합니다.
2. **값(Value)** 입력란에 숫자를 입력합니다.
3. **기준 단위(From)** 드롭다운에서 입력값의 단위를 선택합니다.
4. 우측 결과 카드에 모든 단위로 변환된 값이 표시됩니다.
5. 결과 카드를 클릭하면 값이 복사되고 해당 단위가 새 기준 단위가 됩니다.

---

## 도움말 편집 방법

1. 헤더 우측 **도움말** 버튼 클릭
2. 모달 상단 **✏️ 편집** 버튼 클릭
3. 텍스트·표 셀을 직접 클릭하여 내용 수정
4. **💾 저장 후 다운로드** 버튼 → `index.html` 파일 저장
5. 저장된 파일을 GitHub에 올리면 모든 방문자에게 반영됩니다

---

## GitHub Pages 배포 방법

### 1단계 — Repository 생성

1. [github.com](https://github.com) 에 로그인합니다.
2. 우측 상단 **+** → **New repository** 클릭
3. Repository name 입력 (예: `units-conversion`)
4. **Public** 선택 (Pages는 Public만 무료 제공)
5. **Create repository** 클릭

### 2단계 — 파일 업로드

1. 생성된 Repository 페이지에서 **Add file** → **Upload files** 클릭
2. `index.html`, `README.md`, `_config.yml` 세 파일을 드래그 앤 드롭
3. **Commit changes** 클릭

### 3단계 — GitHub Pages 활성화

1. Repository 상단 **Settings** 탭 클릭
2. 좌측 메뉴 **Pages** 클릭
3. **Source** 섹션에서 Branch: `main` / Folder: `/ (root)` 선택
4. **Save** 클릭

### 4단계 — URL 접속

- 약 1~2분 후 아래 URL로 접속 가능합니다:
  ```
  https://{GitHub 사용자명}.github.io/{Repository 이름}/
  ```
  예: `https://johndoe.github.io/units-conversion/`

### 업데이트 방법

도움말 편집 후 다운로드한 `index.html`을 Repository에 다시 업로드하면  
GitHub Pages가 자동으로 갱신됩니다 (보통 수십 초 ~ 2분 소요).
