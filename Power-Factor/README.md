# ⚡ Power Factor · Web v2.5

역률 개선(Power Factor Correction)을 위한 콘덴서 용량 산정 웹 도구입니다.  
단일 HTML 파일로 구성되어 있어 **서버 없이 브라우저에서 바로 실행** 가능합니다.

---

## 앱 소개

전기 설비 설계 시 역률 개선 콘덴서(커패시터) 용량을 손쉽게 산정할 수 있는 도구입니다.  
유효전력(P)과 공급 피상전력(Ps)을 입력하면 현재 역률을 자동으로 계산하고,  
목표 역률 달성에 필요한 콘덴서 용량(Qc)을 즉시 산출합니다.

---

## 주요 기능

- **역률 자동 계산** — P, Ps 입력 시 현재 cos θ 실시간 산출
- **Auto 모드** — 목표 역률(cos θ') 입력 → 필요 콘덴서 용량(Qc) 자동 계산
- **User 모드** — 설치 가능한 Qc 직접 입력 → 개선 후 역률 역산
- **개선 전·후 비교 테이블** — 피상전력 / 역률 / 무효전력 / Qc를 Before · After · Diff. % 형태로 표시
- **도움말 모달** — 앱 개요, 입력 항목 설명, 계산식, 주의사항 포함
- **인앱 도움말 편집** — 모달 내에서 텍스트·표 셀을 직접 수정 후 index.html로 저장·다운로드

---

## 사용법

> 별도 설치 불필요 — 브라우저에서 `index.html` 파일을 열거나, GitHub Pages URL에 접속하면 바로 실행됩니다.

1. **유효전력 P** (kW)와 **공급 피상전력 Ps** (kVA)를 입력합니다.
2. **산정 방식**을 선택합니다.
   - **Auto**: 목표 역률(cos θ') 입력 → Qc 자동 산출
   - **User**: 설치 가능한 Qc 직접 입력 → 개선 후 역률 역산
3. **검토 실행** 버튼을 클릭하면 개선 전·후 비교 결과가 표시됩니다.
4. 헤더의 **❓ 도움말** 버튼으로 상세 사용 안내를 확인할 수 있습니다.

### 도움말 편집 및 저장

1. **❓ 도움말** → **✏️ 편집** 클릭
2. 텍스트 또는 표 셀을 클릭해 직접 수정
3. **💾 저장 후 다운로드 (index.html)** 클릭 → 수정 내용이 반영된 파일 다운로드
4. 다운로드한 파일을 GitHub에 재업로드하면 모든 방문자에게 반영됩니다.

## 적용 기준

본 도구는 아래 기준에 따른 계산식을 적용합니다.

| 항목 | 적용 식 |
|---|---|
| 현재 역률 | cos θ = P / Ps |
| 무효전력 (개선 전) | Pr = √(Ps² − P²) |
| 피상전력 (개선 후, Auto) | Pa' = P / cos θ' |
| 무효전력 (개선 후, Auto) | Pr' = √(Pa'² − P²) |
| 필요 콘덴서 용량 | Qc = Pr − Pr' |
| 역률 역산 (User 모드) | cos θ' = P / √(P² + (Pr−Qc)²) |

> 참고 기준: IEC 60831, KEC(한국전기설비규정) 관련 역률 개선 권고치

## GitHub Pages 배포 방법

### 1. GitHub에서 새 Repository 생성

1. [github.com](https://github.com) 로그인
2. 우측 상단 **+** → **New repository** 클릭
3. Repository name 입력 (예: `power-factor`)
4. **Public** 선택 후 **Create repository** 클릭

### 2. 파일 3개를 Repository 루트에 업로드

1. 생성된 Repository 페이지에서 **Add file** → **Upload files** 클릭
2. ZIP을 압축 해제한 뒤, 아래 파일 3개를 **폴더째로 올리지 말고 파일만** 드래그하여 업로드합니다:
   - `index.html`
   - `README.md`
   - `_config.yml`
3. 하단 **Commit changes** 클릭

### 3. Repository → Settings → Pages

1. Repository 상단 **Settings** 탭 클릭
2. 좌측 사이드바 **Pages** 클릭

### 4. Source 설정 후 저장

1. **Source** 항목에서 **Deploy from a branch** 선택
2. Branch를 `main` (또는 `master`) 선택 → 폴더는 `/ (root)` 유지
3. **Save** 클릭

### 5. URL 접속

- 1~3분 후 아래 URL로 접속 가능합니다:
  ```
  https://[계정명].github.io/[저장소명]/
  ```
  예시: `https://username.github.io/power-factor/`

> **업데이트 방법**: 앱 내 도움말 편집 후 `index.html`을 다운로드하여 GitHub에 재업로드하면 자동으로 반영됩니다.
