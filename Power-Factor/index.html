계산식: cos θ = P / Pa · Pr = √(Pa² − P²) · 개선 후 Pa' = P / cos θ' , Pr' = √(Pa'² − P²) · 필요 콘덴서 Qc = Pr − Pr'. User 모드에서는 입력한 Qc로 Pr' = Pr − Qc, cos θ' = P / √(P² + Pr'²) 를 역산합니다. 모든 값은 참고용이며 적용 규격으로 검증 필요.
